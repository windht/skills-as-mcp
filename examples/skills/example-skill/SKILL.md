# Example Skill

Use this skill when you want a compact example of the file structure that
`skills-as-mcp` can expose through MCP.

## Included assets

- `references/example-notes.md` for supporting context
- `scripts/preflight.sh` for runnable helper logic

## Guidance

1. Read the `SKILL.md` file first.
2. Inspect related files only when you need deeper context.
3. Keep skill names stable once clients depend on them.
